# 题目描述

给定一个华氏温度**F**，本题要求编写程序，计算对应的摄氏温度**C**。计算公式：$C=5×(F−32)/9$。题目保证输入与输出均在整型范围内。

### 输入格式:

输入在一行中给出一个华氏温度。

### 输出格式:

在一行中按照格式“Celsius = C”输出对应的摄氏温度C的整数值。

### 输入样例:

```in
150
```

### 输出样例:

```out
Celsius = 65
```

# 答案源码

```python
F=int(input())
print(f'Celsius = {int(5*(F-32)/9)}')
```

# 细节讲解

- 注意运算优先级
- 输出为整数，所以要使用int()
