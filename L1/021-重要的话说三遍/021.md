# 题目描述

这道超级简单的题目没有任何输入。

你只需要把这句很重要的话 —— “I'm gonna WIN!”——连续输出三遍就可以了。

注意每遍占一行，除了每行的回车不能有任何多余字符。

### 输入样例：

```in
无
```

### 输出样例：

```out
I'm gonna WIN!
I'm gonna WIN!
I'm gonna WIN!
```
