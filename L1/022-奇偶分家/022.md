# 题目描述

给定`N`个正整数，请统计奇数和偶数各有多少个？

### 输入格式：

输入第一行给出一个正整`N`（**≤**1000）；第2行给出`N`个非负整数，以空格分隔。

### 输出格式：

在一行中先后输出奇数的个数、偶数的个数。中间以1个空格分隔。

### 输入样例：

```in
9
88 74 101 26 15 0 34 22 77
```

### 输出样例：

```out
3 6
```


# 答案源码

## C

```c
#include <stdio.h>
int main(){
    int N,num,j=0,k=0;
    scanf("%d",&N);
    for(int i=0;i<N;i++){
        scanf("%d",&num);
        if (num%2==0){
            k+=1;
        }
        else{
            j+=1;
        }
    }
    printf("%d %d",j,k);
}
```
