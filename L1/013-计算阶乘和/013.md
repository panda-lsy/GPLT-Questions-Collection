# 题目描述

对于给定的正整数**N**，需要你计算 $S=1!+2!+3!+...+N!$。

### 输入格式：

输入在一行中给出一个不超过10的正整数**N**。

### 输出格式：

在一行中输出**S**的值。

### 输入样例：

```in
3
```

### 输出样例：

```out
9
```


# 答案源码

```python
N=int(input())
S=0
for u in range(1,N+1):
    temp=1
    for i in range(1,u+1):
        temp*=i
    S+=temp
print(S)
```

# 细节讲解

- 通过设定一个临时变量`temp`，每次在外层循环赋值1，然后内层循环结束相加。
